# 查委托

#### 介绍
这是Yunzai-Bot云崽机器人插件，需要依赖云崽机器人来实现功能

#### 插件功能
本插件可以查询原神每日委托是否有相关的隐藏成就。<br>
查询指令为：#查委托+委托名（指令中的"+"需要删掉、指令与委托名之间有无空格皆可）

#### 安装教程
V2:放进机器人Yunzai-Bot/lib/example文件夹即可。
V3:放进机器人Yunzai-Bot/plugin/example文件夹即可。
